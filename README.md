🎓 ExamFlow - Online Examination System And Automated Results
ExamFlow is a robust and user-friendly web application developed to automate the examination process. Built with PHP and MySQL, it allows educational institutions to conduct secure, timed, and automated assessments with real-time result generation.

🚀 Features
Student Side
Secure Authentication: User registration and session-based login.

Timed Exams: Real-time countdown timer with auto-submit functionality.

Anti-Cheat System: Window focus tracking to prevent tab-switching during exams.

Instant Results: Automated grading and performance overview after submission.

Admin Side
Dashboard: Overview of total students, subjects, and exams.

Content Management: Add/Edit subjects and manage Question Banks.

Bulk Upload: Import questions quickly using CSV files.

Result Analytics: Track and export student scores and performance reports.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (AJAX/jQuery)

Backend: PHP (Procedural)

Database: MySQL

Server: XAMPP / WAMP / Apache

💻 Installation & Setup
Clone the Repository:

Bash
git clone https://github.com/your-username/ExamFlow.git
Move to Server Folder:
Copy the ExamFlow folder to your htdocs (XAMPP) or www (WAMP) directory.

Database Setup:

Open phpMyAdmin (http://localhost/phpmyadmin).

Create a new database named exam_db.

Import the database.sql file provided in the repository.

Configuration:

Open config.php and update your database credentials if necessary.

Run Application:

Open your browser and go to http://localhost/ExamFlow.
